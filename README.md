# AQB-FAS

Project scaffold for Adaptive Quality-Bitrate Face Anti-Spoofing (AQB-FAS).

Structure:
- configs/: experiment configs (YAML)
- data/: raw (not committed) and processed CSVs
- src/: code (datasets, models, training, eval)
- scripts/: utility scripts
- outputs/: runs, tables, figures
- paper/: LaTeX source

See `requirements.txt` for dependencies.
